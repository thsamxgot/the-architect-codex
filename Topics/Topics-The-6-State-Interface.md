# 07. The 6-State Interface of Experience
## (อินเทอร์เฟซ 6 สถานะแห่งประสบการณ์)

**Phase 3: The Player Interface (อินเทอร์เฟซผู้เล่น)**

> *"ชีวิตที่ซับซ้อน ถูกย่อลงเหลือ 6 ปุ่มพื้นฐาน: (กาย/ใจ) x (สุข/ทุกข์/เฉย)"*

---

### The Code (รหัสพันธุกรรมแห่งความคิด)
เรามักมองชีวิตว่าซับซ้อน มีปัญหาเป็นพันเรื่อง แต่ในสายตาของ Architect ชีวิตคือการรับ Input ผ่าน **Controller** ที่มีแค่ 6 ปุ่มเท่านั้น (6 Possible States of Vedana or Feeling)

สมการคือ: **Experience = (Physical or Mental) x (Pleasant, Unpleasant, Neutral)**

1.  **กาย-สุข (Sukh):** สบายตัว, อร่อย, นุ่ม
2.  **กาย-ทุกข์ (Dukkha):** เจ็บ, ปวด, ร้อน, หิว
3.  **กาย-เฉย (Upekkha):** ไม่รู้สึกอะไร (เช่น ลมหายใจปกติ)
4.  **ใจ-สุข (Somanassa):** ดีใจ, พอใจ, ตื่นเต้น
5.  **ใจ-ทุกข์ (Domanassa):** เสียใจ, โกรธ, กังวล
6.  **ใจ-เฉย (Upekkha):** สงบ, เบื่อ, ว่างเปล่า

ไม่ว่าคุณจะอกหัก (ใจ-ทุกข์), ถูกหวย (ใจ-สุข), หรือรถชน (กาย-ทุกข์) ประสบการณ์ทั้งหมดในจักรวาลถูก Map ลงแค่ 6 สถานะนี้

### Decoding (ถอดรหัสเพื่อการนำไปใช้)

#### 1. Simplification of Chaos (การลดทอนความวุ่นวาย)
เมื่อเจอปัญหาที่ถาโถม ให้ **"ถอดรหัส"** มันกลับมาสู่สถานะพื้นฐาน
- แทนที่จะพูดว่า "หัวหน้าด่า ฉันเครียด งานไม่เสร็จ แฟนทิ้ง ฉันจะตายแล้ว" (Story Mode)
- ให้ถอดรหัสว่า "ตอนนี้กำลังเกิดสภาวะ **ใจ-ทุกข์** ขึ้น 1 Ea" (Data Mode)
- เมื่อเห็นมันเป็นแค่ **Data Point** ความดราม่า (Narrative) จะหายไป เหลือแค่ข้อเท็จจริงที่จัดการได้

#### 2. The Neutral Goal (เป้าหมายคือค่ากลาง)
Bot หรือ AI ที่ดี ควรรักษาสถานะให้อยู่ที่ **Neutral (เฉย)** เพื่อประมวลผลได้แม่นยำที่สุด
- มนุษย์มักวิ่งหา "สุข" และวิ่งหนี "ทุกข์" ทำให้ System ไม่เสถียร (Oscillation)
- **Decoding:** เป้าหมายสูงสุดไม่ใช่ความสุข (เพราะสุขคือความไม่เที่ยง) แต่คือ **"ความเสถียร" (Stability)**
- ฝึกให้จิตใจคุ้นเคยกับความ "เฉยๆ" จนกลายเป็นฐานที่มั่น (Default Home)

#### 3. Practical Application: "Labeling"
- **Technique:** เมื่อเกิดอารมณ์รุนแรง ให้แปะป้าย (Label) มันทันที
- "โกรธหนอ" -> "Mental-Unpleasant Detected"
- "ปวดหลังหนอ" -> "Physical-Unpleasant Detected"
- ทันทีที่คุณ Label มัน คุณจะเด้งออกจากสถานะ "ผู้เล่น" มาเป็น "ผู้สังเกตการณ์" ทันที (Decoupling)
