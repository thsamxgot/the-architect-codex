# 22. The Mortality Function: The Value of Scarcity
## (ฟังก์ชันแห่งความตาย: มูลค่าของความขาดแคลน)

**Phase 7: The Final Sequence (บทสรุป)**

> *"ถ้าทองคำมีอยู่ทุกที่ ทองคำจะไร้ค่า ถ้าชีวิตเป็นอมตะ ชีวิตจะไร้ความหมาย"*

---

### The Code (รหัสพันธุกรรมแห่งความคิด)
ความตาย (Death) คือฟีเจอร์ที่สำคัญที่สุดของระบบชีวิต
ลองจินตนาการถึงเกมที่คุณเป็นอมตะ (God Mode)
- แรกๆ สนุก
- ผ่านไป 3 วัน น่าเบื่อ
- ผ่านไป 1 ปี ทรมาน
- ถ้าไม่มี Deadline ก็ไม่มีแรงกระตุ้น (Urgency) ที่จะทำอะไรเลย

**Death defines Life.** ความตายไม่ได้อยู่ตรงข้ามกับชีวิต แต่เป็นส่วนหนึ่งของชีวิต เหมือน "ขอบ" ของแก้ว ที่ทำให้น้ำมีรูปทรง

### Decoding (ถอดรหัสเพื่อการนำไปใช้)

#### 1. Scarcity Creates Value (ความขาดแคลนสร้างมูลค่า)
เวลาของเรามีจำกัด (Limited Resource) ทำให้ทุกวินาทีมีค่ามหาศาล
- การที่คุณอ่านข้อความนี้อยู่ แลกมาด้วย "วินาทีชีวิต" ที่คุณไม่มีวันได้รับคืน
- เมื่อตระหนักถึงสิ่งนี้ คุณจะเลิกเสียเวลากับเรื่องไร้สาระ (Drama, Hate, Worry) โดยอัตโนมัติ

#### 2. The Final Deadline (เส้นตายสุดท้าย)
คนส่วนใหญ่ใช้ชีวิตเหมือนตัวเองเป็นอมตะ (ผัดวันประกันพรุ่ง)
- Architect ต้องมองเห็น **Count Down Timer** บนหัวตัวเองเสมอ
- ไม่ใช่เพื่อให้กลัว แต่เพื่อให้ **"ตื่น"**
- ถามตัวเอง: "ถ้าวันนี้คือวันสุดท้าย Code ที่ฉันเขียน (ชีวิตที่ฉันใช้) พร้อมส่งมอบ (Submit) หรือยัง?"

#### 3. Practical Application: "Memento Mori"
- ระลึกถึงความตายวันละครั้ง
- ไม่ใช่คิดว่า "ฉันกำลังจะตาย" (หดหู่)
- แต่คิดว่า **"ฉันยังมีชีวิตอยู่!"** (ขอบคุณโอกาส)
- เปลี่ยนความกลัวตาย เป็นพลังในการใช้ชีวิตอย่างเข้มข้น (Intensity of Living)
