# Note 22: Particle Identity Blueprint

**Subject:** Why are all Electrons identical?
**Reference:** The Architect's Codex - Topological Identity

---

## 1. Topology Defines Type
*   **Rule:** Particles are defined by **Integer** complexity (Number of twists).
*   **Identity:** All electrons share the exact same Topological Blueprint (e.g., Knot Type $3_1$).
*   **Discreteness:** You cannot have a "1.5 Twist". It's either 1 (Electron) or higher (Muon).

## 2. State Variables
While Identity is fixed, **State** varies:
*   **Kinetic Energy:** Vibration of the host strand.
*   **Position:** Coordinate in the mesh.
*   **Spin Direction:** Up/Down orientation.

## 3. The Knot Analogy
*   A "Square Knot" is identical regardless of where it is tied.
*   If you add an extra loop, it's no longer a "Square Knot" (It becomes a different particle).
*   Therefore, all electrons have identical Mass/Charge because they correspond to the exact same geometric knot equation.
