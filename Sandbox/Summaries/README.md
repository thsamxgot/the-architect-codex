# Sandbox Summaries

This directory contains the **Master Summaries** for the extracted research from "The Architect's Codex".

These summaries represent the "Single Source of Truth" for distinct topics, consolidated from the various notes, research files (Rxx), and theoretical papers (Txx) found in the parent directories.

## Contents

The summaries are organized into 14 core topics, ranging from the metaphysical origins of the universe to the hard physics of the Strand Model and the system architecture of reality.

## Goals of these Summaries

1.  **Consolidation:** Merge scattered information from multiple source files into cohesive narratives.
2.  **Clarity:** Use clear, consistent terminology (e.g., distinguishing "Will" vs. "Consciousness").
3.  **Completeness:** Ensure every major concept from the Codex is represented.
